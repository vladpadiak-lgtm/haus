# HAUS

Двомовний українсько-словацький каталог із 40 концепціями приватних будинків для будівництва у Словаччині.

## Що є на сайті

- 40 проєктів із власними зображеннями, площею, бюджетом і характеристиками;
- 80 інтер’єрних ракурсів — по два унікальні дизайн-концепти всередині кожного будинку;
- окремий концептуальний план поверху для кожного будинку із зонуванням і площами приміщень;
- повний переклад інтерфейсу, карток і деталей словацькою з перемикачем UA / SK;
- сортування за ціною та площею;
- фільтри за бюджетом, площею, кімнатами, поверховістю, технологією та енергокласом;
- детальна картка кожного будинку з матеріалами, структурою кошторису та строками;
- адаптивний інтерфейс для комп'ютерів і телефонів.

## Локальний запуск

Сайт не має зовнішніх залежностей. Відкрийте `index.html` у браузері або запустіть будь-який статичний HTTP-сервер у корені репозиторію.

## Про ціни

Діапазони є орієнтовними сценаріями для Словаччини, а не комерційною пропозицією. Земля, меблі, кухня, благоустрій, складні зовнішні мережі та нестандартні роботи не включені. Перед будівництвом потрібні прив'язка до ділянки, інженерні розрахунки й пропозиції місцевих підрядників.
